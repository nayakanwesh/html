HISTORY OF W3C

In 1989, Tim Berners-Lee
invented the World Wide Web (see the original proposal).
He coined the term "World Wide Web," wrote the first World Wide Web server, "httpd," and the first client program (a browser and editor), "WorldWideWeb," in October 1990.
He wrote the first version of the "HyperText Markup Language" (HTML),
the document formatting language with the capability for hypertext links that
became the primary publishing format for the Web. His initial specifications
for URIs, HTTP, and HTML were refined and discussed in larger circles as Web technology spread.
